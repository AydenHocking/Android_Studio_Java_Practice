# Android Studio Java Practice

This is a collection of practice programs I created in Android Studio using Java to demonstrate different uses and scenarios. Each file is its own program individually.

## Backend:
- `.java` Java v.17.0.13
- `.csv` Data
- `.xml` res and Manifest
- Additional Files are Included

## Tools:
- Git and GitHub for version control
- Postman for database viewing
- Android Studio for local editing
- Firestore for web viewing
